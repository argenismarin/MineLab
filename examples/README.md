# Ejemplos de MineLab

Scripts de ejemplo que demuestran flujos de trabajo completos.

| Script | Descripción |
|--------|-------------|
| `01_geostatistics_workflow.py` | Sondaje → variograma → kriging → recurso |
| `02_economic_evaluation.py` | Cashflow → NPV → Monte Carlo → sensibilidad |
| `03_blast_design.py` | Patrón → fragmentación → vibración → seguridad |
| `04_mineral_processing.py` | Conminución → flotación → balance metalúrgico |
| `05_slope_stability.py` | RMR → Hoek-Brown → Bishop → soporte |
| `06_ventilation_network.py` | Resistencias → Hardy Cross → ventilador |
| `07_pit_optimization.py` | Block model → LG 2D → scheduling → NPV |
| `08_complete_mine_workflow.py` | Flujo end-to-end completo |
| `09_utilities_reference.py` | Conversiones, minerales, grados y estadística |
| `10_equipment_fleet.py` | Ciclo de camiones, flota y productividad |
| `11_production_blending.py` | Blending, control de leyes, stockpiles y reconciliación |
| `12_underground_mining.py` | Caserón → convergencia-confinamiento → relleno |
| `13_hydrogeology.py` | Ensayo de acuífero → desaguado → calidad de agua |

## Ejecución

```bash
py -m uv run python examples/01_geostatistics_workflow.py
```

MineLab Documentation
=====================

**Librería integral de ingeniería minera y metalúrgica para Python.**

MineLab proporciona herramientas de cálculo para todas las disciplinas de la
ingeniería de minas: geoestadística, planeamiento minero, geomecánica,
procesamiento de minerales, perforación y voladura, ventilación, economía
minera, medio ambiente y más.
